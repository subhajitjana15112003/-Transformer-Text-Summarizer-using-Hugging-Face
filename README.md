# AI Text Summarizer using T5 Transformer and FastAPI

An AI-powered Text Summarization web application built using Hugging Face Transformers, PyTorch, and FastAPI.  
This project uses the T5 Transformer model to generate concise summaries from long dialogues or text inputs.

---

## Features

- Transformer-based text summarization using T5
- FastAPI backend for high-performance APIs
- Interactive web interface using HTML templates
- Automatic CPU/GPU device detection
- REST API support for integration
- Clean text preprocessing pipeline
- Real-time summarization

---

## Tech Stack

- Python
- FastAPI
- PyTorch
- Hugging Face Transformers
- Jinja2
- HTML/CSS
- Uvicorn
